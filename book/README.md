# The Rescue Boat — Long-Form Manuscript

**Subtitle:** Survival, Shame, and the Language We Needed

This folder holds the **long-form book build** (Option A). It is separate from the public **12-article counter-narrative site** at [daniel-lingar.github.io/rescue-boat](https://daniel-lingar.github.io/rescue-boat/).

## Two Tracks

| Track | Location | Purpose |
|-------|----------|---------|
| **Site** | `client/src/content/article_*.md` | Short counter-narratives (live on GitHub Pages) |
| **Book** | `book/manuscript/` | Full ~85k-word manuscript per `MASTER_BLUEPRINT.md` |

The articles are source material and voice reference. The book expands them into 17 chapters + preface + introduction + appendix.
